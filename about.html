About the Database System

The database system implemented by Solomon Kennedy Drzamado serves as a comprehensive solution for managing hospital operations. This software engineering project was developed using Python, HTML, CSS, and JavaScript.

System Development

The first version of the system was built using Python. Subsequently, an enhanced version was developed using Python with the Flask framework for the backend, and HTML, CSS, and JavaScript for the frontend.

Project Details

The project aims to streamline hospital management processes, including patient management, doctor scheduling, pharmacy inventory, and administrative tasks. It demonstrates Solomon's familiarity with software development and database management for healthcare institutions.
